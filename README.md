# Student Profile App

A mobile app built with Expo and React Native to manage student profiles.

## About
This app allows you to add, view and manage student profiles including name, age, course, roll number, email and profile picture.

## Features
- Add student profiles
- View all students in a list
- Auto-generated profile avatars
- Clean and simple UI

## Tech Stack
- Expo (React Native)
- JavaScript/TypeScript
- React Hooks (useState)
